Sobre Mim

Olá! Eu sou o João, um desenvolvedor apaixonado por resolver problemas através de código. Minha jornada na programação começou com o desejo de construir coisas úteis, e hoje tenho o prazer de trabalhar com tecnologias de ponta para criar aplicações web eficientes.

Minha principal ferramenta é o Go (Golang), pela sua simplicidade, performance e ecossistema robusto para o desenvolvimento backend. Tenho experiência sólida na construção de APIs RESTful, arquitetura de microsserviços e na integração com diversos bancos de dados.

Aqui estão algumas tecnologias com as quais tenho trabalhado recentemente:

  • Go (Golang)
  • Docker & Kubernetes
  • PostgreSQL
  • gRPC
  • Templ & htmx
  • Gin / Fiber
